  • transition-duration : 요소의 애니메이션이 완료되데 걸리는 시간을 설정합니다.
  • transition-property : 전환 효과를 적용할 css속성을 설정합니다.
  • transition-timing-function : 전환 효과의 영향의 중간값의 계산을 설정합니다.
  • translate : 번역 변형을 설정합니다.
  • unicode-bidi : 문서의 텍스트가 양방향으로 처리되는 방식을 설정합니다.
  • user-select : 사용자가 텍스트를 선택할수 있는지의 여부를 설정합니다.
  • vertical-align : 인라인 또는 테이블 셀 박스의 수직 정렬을 설정합니다.
  • visibility : 레이아웃을 변경하지 않고 요소를 보이거나 숨깁니다.
  • white-space : 공백 문자의 처리 방법을 설정합니다.
  • widows : 페이지 맨위에 표시되는 블록 컨테이너의 최소 수를 설정합니다.
  • width : 요소의 넓이를 설정합니다.
  • will-change : 요소에 예상되는 변화 종류에 관한 힌트를 브라우저에게 제공합니다.
  • word-break : 텍스트가 콘텐츠 밖으로 오버풀로 할때 줄 바꿈을 설정합니다.
  • word-spacing : 단어와 단어 사이, 태그와 태그 사이의 거리를 설정합니다.
  • writing-mode : 텍스트의 가로, 세로의 배치여부와 진행방향을 설정합니다.
  • z-index : 위치 지정요소와 하위 플렉스 아이템의 z축 순서를 설정합니다.

searchKeyword.forEach((el) => {
    el.addEventListener("click", () => {
        const searchWord = el.innerText; //사용자가 클릭한 알파벳

        // console.log(searchWord);

        searchList.forEach((el) => {
            const cssName = el.querySelector("strong").innerText; //속성 텍스트
            // console.log(cssName)
            // searchNum.textContent = cssName.length;
            //알파벳 첫글자 == CSS 속성의 첫글자
            if (searchWord === "All") {
                //All버튼을 클릭했을때
                el.classList.remove("hide");
            } else if (searchWord.charAt(0) === cssName.charAt(0)) {
                el.classList.remove("hide");
            } else {
                el.classList.add("hide");
            }
        });
        const searchCount = document.querySelectorAll(
            ".search__list ul .hide"
        );
        searchInfo.textContent =
            searchWord +
            "의 갯수는" +
            (searchList.length -= searchCount.length) +
            "개 입니다^^!";
    });
});
