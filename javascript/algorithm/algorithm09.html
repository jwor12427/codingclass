x만큼 간격이 있는 n개의 숫자

문제 설명

함수 solution은 정수 x와 자연수 n을 입력 받아, x부터 시작해 x씩 증가하는 숫자를 n개 지니는 리스트를 리턴해야 합니다. 다음 제한 조건을 보고, 조건을 만족하는 함수, solution을 완성해주세요.

제한 조건

😎x는 -10000000 이상, 10000000 이하인 정수입니다.
😎n은 1000 이하인 자연수입니다.

입출력 예시

x n answer
2 5 [2,4,6,8,10]
4 3 [4,8,12]
-4 2 [-4, -8]

😎내가 한 풀이

처음에 answer에다가 +=로 추가하면 될줄 알았는데... 배열로 만들어야 해서 push를 써야한다는 것을 뒤늦게 깨달았다...! ㅠㅠ 

const x = 2;
const n = 5;
function solution(x, n) {
    var answer = [];
    for(let i = 1; i <= n; i++){
        answer.push(x*i);
    }
    return answer;
}

😎다른 풀이

fill이라는 메서드는 처음 보았다...

Array.fill()
😎메서드는 배열의 시작 인덱스부터 끝 인덱스의 이전까지 정적인 값 하나로 채웁니다.
arr.fill(value[, start[, end]])
value: 배열을 채울 값
start: 시작 인덱스, 기본 값은 0
end: 끝 인덱스, 기본 값은 this.length 

function solution(x, n) {
    return Array(n).fill(x).map((v, i) => (i + 1) * v)
}
