나머지가 1이 되는 수 찾기

문제 설명

자연수 n이 매개변수로 주어집니다. n을 x로 나눈 나머지가 1이 되도록 하는 가장 작은 자연수 x를 return 하도록 solution 함수를 완성해주세요. 답이 항상 존재함은 증명될 수 있습니다.

제한 조건

🐾3 ≤ n ≤ 1,000,000

입출력 예시

n result
10 3
12 11

🐾내가 한 풀이

if문에서 나누었을때 나머지가 1이 되는 수를 return했당... 

const n = 10;
function solution(n) {
    for(let i = 0; i < n; i++){
        if(n%i === 1){
            return i;
        }
    
    }
}

🐾다른 풀이

while문도 사용 가능

function solution(n, x = 1) {    
    while (x++) {
        if (n % x === 1) {
            return x;
        }
    }    
}
